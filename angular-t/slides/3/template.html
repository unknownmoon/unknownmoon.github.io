What are scopes in AngularJS

Scopes are objects that

  • provide context against which expressions (such as {{username}}) are evaluated.
  • provide APIs ($watch) to observe model mutations.
  • provide APIs ($apply) to propagate any model changes through the system into the view from outside of the "Angular realm" (controllers, services, Angular event handlers).
  • provide APIs to propagate ($broadcast, $emit) or catch ($on) Angular events.
  • can be nested to isolate application components while providing access to shared model properties. A scope (prototypically) inherits properties from its parent scope.
$rootScope, service provides the global scope for Angular.
$scope, service provides the local scope for Angular.
  • How to retrieve the scope attached to particular element
    • You can use angular.element(DOM_ELEMENT).scope() to retrieve the associated $scope (not recommended in application, but for debugging).
