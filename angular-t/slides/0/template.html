Expressions are JavaScript-like code snippets that are usually placed in bindings such as {{ expression }}. Expressions are processed by the $parse service. Expressions are often post processed using filters to create a more user-friendly format.

For using filters <to be covered later>, you can use binding such as {{ expression | filter }} to pass the result of your expression to a filter; or use {{ expression | filter1 | filter2 | ... }} to pipe the filters; even pass arguments to filter by using {{ expression | filter:argument1:argument2:... }}
